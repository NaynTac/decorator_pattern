# Лабораторная работа №2
### Содержимое проекта:

- **data/** содержит примеры файлов, полученных при выполнении методов декораторов (*csv* и *yaml* файлы)
- **.gitignore** содержит информацию о файлах и каталогах, которые не будут отправляться на удаленный репозиторий
- **.python-version** содержит информацию об используемой версии Python
- **main.py** содержит основной функционал программы
- **test_main.py** содержит тесты
- **uv.lock** содержит информацию обо всех зависимостях в проекте
- **pyproject.toml** содержит основную информацию о проекте:
	- Название
	- Версия
	- Описание
	- Имя README-файла
	- Версия Python
	- Зависимости (включая версии)
___
### Краткое описание функционала программы

Программа содержит 5 классов:
- **CurrenciesABC**. Абстрактный класс для описания изменяемых методов
- **Currencies**. Базовый класс со следующим функционалом:
	- *get_currencies*. Метод для получения данных о курсах валют, полученных с помощью API Центробанка. Возвращает словарь с полученными данными в формате JSON
- **Decorator**. Базовый класс декоратора класса **Currencies**
- **CurrenciesDecoratorCSV**. Декоратор класса **Currencies** со следующим измененным функционалом:
	- *get_currencies*. Метод, использующий значение метода базового класса для записи полученных данных в CSV-файл
- **CurrenciesDecoratorYAML**. Декоратор класса **Currencies** со следующим измененным функционалом:
	- *get_currencies*. Метод, использующий значение метода базового класса для записи полученных данных в YAML-файл
___
### Краткое описание тестов

Всего в проекте содержится 3 теста:
- **test_currencies**. Тест проверяет, что метод *get_currencies* базового класса **Currencies** возвращает корректные значения
- **test_currencies_csv**. Тест проверяет, что открытие файла и запись в него данных при вызове измененного метода *get_currencies* декоратора **CurrenciesDecoratorCSV** прошла корректно с нужными аргументами
- **test_currencies_yaml**. Тест проверяет, что открытие файла и запись в него данных при вызове измененного метода *get_currencies* декоратора **CurrenciesDecoratorYAML** прошла корректно с нужными аргументами
